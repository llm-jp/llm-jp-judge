## Installation

```
python3 -m venv venv
source venv/bin/activate
pip install -r requrements.txt
```

## Setup

評価に使用するAPIの情報を`.env`ファイルに入力して下さい。  
(評価にvLLMを用いる場合は必要ありません。)

### Azure OpenAI API

```bash
AZURE_ENDPOINT="https://********.openai.azure.com/"
AZURE_API_KEY="********"
```

### AWS Bedrock API

```bash
AWS_ACCESS_KEY="********"
AWS_SECRET_KEY="****************"
AWS_REGION="us-west-2"
```
